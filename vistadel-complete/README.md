## What is VISTADEL?

VISTADEL is a **8-phase app development framework** designed to help teams ship production-grade applications faster, safer, and with less guesswork.

Built on real client work from INT Inc., tested across professional services, healthcare, SaaS, eCommerce, and manufacturing verticals.

**V**ision → **I**ntegration → **S**tack → **T**esting → **A**I Implementation → **D**eployment → **E**volution → **L**aunch

---

## Three Ways to Use VISTADEL

### **Tier 1: Internal (INT Team)**
For INT sales, pre-sales, and project teams.  
**Location:** Private GitHub branch + gated Notion dashboard  
**Purpose:** "Here's how we build. Here's our process."

### **Tier 2: Client-Facing (INT Clients)**
For clients onboarding with the Customization Service.  
**Location:** Shared Notion workspace + fillable Google Docs  
**Purpose:** "Here's how to work with us. Here's what success looks like."

### **Tier 3: Public (Community)**
For founders, engineers, teams shipping apps.  
**Location:** Public GitHub + blog + video library  
**Purpose:** "Here's how to ship fast with modern tools. Here's the framework we use."
